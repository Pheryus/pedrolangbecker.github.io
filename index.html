Crona

Crona é um jogo de de puzzle onde você ajuda uma entidade chamada Crona, criando padrões de cores vermelho-verde-azul.

Eu programei o jogo e criei um editor de fases para tornar o processo de criação de fases mais fácil. Todos os puzzles e mecânicas foram criados por mim.

Jogo feito no Unity, lançado em Maio 2019 como jogo beta na Google Play Store.

Father's Home

Father's Home é um jogo feito por mim e Vinícius Mateus Dreifke para a Global Game Jam de 2019. Tivemos 48 horas para criar um jogo com o tema "O que lar significa para você?".

Fui responsável pela programação, e ambos trabalhamos no design do jogo.

Father's Home é um escape room 3D, em que você tenta descobrir um segredo em uma sala virtual criada pelo seu falecido pai. A mecânica principal do jogo é o uso de uma habilidade "hacker": você consegue enxergar o código da maioria das coisas do mundo virtual clicando com o botão direito do mouse. Alguns puzzles só podem ser responsivos dessa forma.

Slender Boardgame

Feito em conjunto comMaurício Pergher Soares, baseado no jogo digital Slender: The Eight Pages. Ambos trabalhamos no game design do jogo.

Slender Boardgame é um jogo cooperativo para dois jogadores, onde você e seu amigo procuram por papéis na floresta enquanto tentam escapar do assustador Slender. Mova os personagens pelo cenário para explorar e procurar por Pistas. Usando as pistas que encontrar, você pode movê-las pelo Tabuleiro de Pistas, levando-as a diferentes símbolos, que garantem bônus e/ou penalidades de acordo com o tamanho que você escolher. Movendo as pistas para locais específicos do Tabuleiro de Pistas, você pode encontrar o papel que precisa.

Feita em um mês como trabalho de Game Design na Universidade Franciscana (UFN).

Você pode ver e baixar o Manual de Regras abaixo.

Drill N' Steal

Drill N' Steal é um platformer 2D side-scrolling platformer estilo 8-bit, em que você é um ladrão que cava buracos para conseguir entrar no cofre de um banco e roubar algum tesouro!

Trabalhei primariamente na programação e desenvolvendo o jogo, ajudando com as mecânicas e level design, além de ter composto a música.

Feito no Unity, ea Game Jam local em 2017.

Factory Fight

Factory Fight é um platformer 2d multiplayer local para dois ou quatro jogadores. Cada jogador controla um robô, que pode pular e interagir com os outros robôs usando armadilhas espalhadas pelo cenário. A mecânica principal do jogo é a energia: os robôs podem energizar o chão abaixo deles, mudando sua cor. Então, eles podem ativar uma armadilha elétrica, fazendo com que todo chão que seja de sua cor eletrifique outros robôs que tocarem nele.

Fui um dos dois programadores, além de participar no game design do jogo.

Feito no Unity para uma gamejam local em 2017.

É necessário ao menos um controle para jogar (dois jogadores).

Microcosmos

Microcosmos é um serious game sobre a estrutura e função pulmonar, em que o jogador controla um personagem que viaja desde a faringe até os pulmões.

Feito com Vinícius Mateus Dreifke para uma cadeira de Projeto de Software na Universidade Federal de Santa Maria (UFSM) em 2016. Usamos a biblioteca pygame do Python para fazer o jogo.

Como a biblioteca pygame não tem uma interface para criar fases, nós criamos um script que cria o mapa do jogo a partir de um arquivo de texto. Outras funcionalidades, como animações de sprite, também foram implementadas no jogo por nós.

Infelizmente, o jogo só roda no Linux. Se quiser testar, você terá que instalar a biblioteca pygame e rodar o comando no terminal python3 start.py dentro da pasta microcosmos/engine.

Plague Siege

Plague Siege é um tower defense, onde o jogador invoca lutadores para proteger sua cidade da Peste Negra. Pulgas e ratos mortais são seu principal inimigo. Há 3 tipos de lutadores com diferentes habilidades que podem até ser aprimorados! Proteja a cidade matando todas criaturas, não deixando uma única invadir a cidade.

Feita para a Globam Game Jam de 2018, usando o Unity.

Nightmares and Heavens

Nightmares and Heavens é um platformer 2D com elementos de puzzle. O jogador controla uma pequena garota que acorda tarde da noite assustada de um pesadelo, se protegendo apenas com seu cobertor. Ela tenta entrar no quarto de seus pais, mas ninguém responde.

O jogador controla a garota por todo gameplay. Ela pode se mover para a esquerda ou direita, escalar, pular e puxar pequenos objetos, além de poder usar seu cobertor. Ela pode arremessra seu cobertor para cobrir objetos ou criaturas, além de colocá-lo em si mesma para se proteger.

Feito para uma game jam local em 2019.

Mono Chameleon

Em Mono Chameleon, você controla um camaleão que consegue enxergar somente um tipo de cor por vez. Sempre que se move para uma direção, você enxerga objetos de uma cor diferente. Você deve ser rápido, se movendo pelos locais certos, prestando atenção em quais aparecem de acordo com a cor que está enxergando.

Participei da programação e game design, além de ter sido responsável pelo audio e música do jogo. Utilizei o middleware FMOD para controlar a progressão de música, bem como os efeitos sonoros.

Feito para uma game jam local em 2019, no Unity.
